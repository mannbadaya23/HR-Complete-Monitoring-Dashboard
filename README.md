# HR Analytics Dashboard | Excel

Interactive HR analytics dashboard developed in Microsoft Excel to analyze employee demographics, workforce distribution, salary insights, leave tracking, and performance metrics through dynamic visual reporting.

## Project Overview

This dashboard provides a centralized HR reporting solution for monitoring employee performance, department distribution, work locations, salary allocation, employment status, and workforce demographics. The project focuses on improving HR visibility and simplifying employee data analysis through interactive dashboards and KPI reporting.

## Key Features

* Interactive KPI cards for:

  * Total Employees
  * Performance Rating
  * Salary Analysis
  * Employment Status
  * Gender Distribution
  * Leave Tracking

* Dynamic visualizations including:

  * Skills Breakdown Analysis
  * Age Group Distribution
  * Department-wise Insights
  * Regional Workforce Analysis
  * Salary Distribution by Job Roles
  * Employment Status Breakdown

* Interactive filtering using:

  * Gender
  * Department
  * Region
  * Employee Categories

## Dashboard Impact & Optimization

* Improved HR data accessibility through centralized employee analytics and interactive reporting.
* Reduced manual HR reporting effort by consolidating workforce, salary, and performance insights into a single dashboard.
* Enhanced analytical visibility using dynamic filters, KPI cards, and comparative workforce analysis.
* Optimized dashboard layout and visual hierarchy for improved usability, reporting clarity, and faster decision-making.
* Enabled efficient monitoring of employee demographics, leave trends, department distribution, and workforce performance metrics.
